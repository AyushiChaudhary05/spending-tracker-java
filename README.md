# Spending Tracker (Java)

A console-based Java application to track daily expenses.

## Features
- Add daily expenses
- View all expenses
- Calculate total spending
- Menu-driven console application

## Technologies Used
- Core Java
- OOP Concepts
- Collections (ArrayList)

## How to Run
```bash
javac *.java
java Main

1. Clone the repository:
   git clone https://github.com/AyushiChaudhary05/spending-tracker-java.git

2. Navigate to the project directory:
   cd spending-tracker-java

3. Compile the project:
   javac *.java

4. Run the application:
   java Main
